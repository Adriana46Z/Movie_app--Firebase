Movie_App

Descriere
Aplicație web pentru gestionarea filmelor cu sistem de filtrare avansată. Utilizatorii își pot face cont. Apoi, aceștia pot căuta filme după actor, an, gen și își pot salva filmele preferate. De asemenea, pot adăuga review uri filmelor și le pot oferi scoruri acestora.
